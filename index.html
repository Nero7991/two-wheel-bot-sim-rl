Two-wheel Balancing Bot Environment

GitHub Repository

Welcome to an interactive machine learning training simulator that demonstrates reinforcement learning (reward based) in action.

Train a neural network to balance a two-wheeled robot directly in your browser. This simulator uses Deep Q-Network (DQN) reinforcement learning with optional WebGPU acceleration for fast training. Export trained models as C++ code for real hardware deployment on microcontrollers like the STM32WL55.

Successfully tested on an Adeept Self-Balancing Robot Kit with firmware available at github.com/Nero7991/two-wheel-bot-firmware.

Watch a neural network learn to balance a two-wheeled robot from scratch, similar to a self-balancing hoverboard or Segway as the episodes go on.

Manual Control

Take Control Click "Enable User Control Only" to disable the controller, then use arrow keys or on-screen buttons to manually balance the robot. Experience how challenging it is!

Train a Model

Step 1 Click "Start Training" to begin machine learning
Step 2 The model will fail repeatedly at first — this is normal
Step 3 After 100-500 episodes, you'll see improvement in the reward chart
Pro Tip: Click "Parallel" to speed up training using multiple CPU core (or workers/threads). This can train faster. Use slower speeds to observe it learn.

Test or Import Models

Test Trained Model After training, click "Test Model" to see your model in action
Import Model Use "Load Model" in Model Management to import a pre-trained model. The GitHub repository includes three pretrained models: Bad, Good, and Great for testing different performance levels.

Default Settings

The default settings are optimized for quick learning. No adjustments needed unless you want to experiment.

What to Expect

• First 100 episodes: Complete chaos — the robot will fall immediately

• Episodes 100-500: Gradual improvement — brief moments of balance, although each session is random. Sometimes it might learn earlier, other times take longer

• Episodes 500+: Stable balancing — the model has learned
